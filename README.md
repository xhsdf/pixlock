# Screen locker script

### Features
-pixelated desktop  
-lock image  
-supports multiple monitors


### Usage

pixlock [file]


If [file] is not specified, a default lock image will be used

### Preview

![preview](https://github.com/xhsdf/pixlock/raw/master/preview.png)

### Requirements
-scrot  
-imagemagick (convert + identify)  
-i3lock
